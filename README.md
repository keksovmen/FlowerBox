# Множество различных проектов под ESP32/8266
Их всех объединяет механизм работы в сети, подключение, сохранение.
Общая архитектура с разделением: представления (.*box_obj.cpp), железа (.*hw_obj.cpp) и их соединения (.*mapper_obj.cpp).
Некоторые пуллят сервера для данных.
Есть приложение для управления на Kotlin под Android: пока приватная репозитория ибо проект лежит в папке учебы, где за 5 лет много документов, необходимо разделить 

- Микро теплица папка project
- Диплом (DMX) папка project2
- Реле папка relay_project
- Управление китайскими игрушками sticks_project
- Коробка на аккумуляторе с дверью на замке и MQTT управлении, [ссылка на документацию](projects/tressure/README.md)
